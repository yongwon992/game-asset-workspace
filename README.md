# Game Asset Workspace

게임용 이미지 에셋을 업로드하고 편집한 뒤,
생성형 AI 요청 과정을 체험할 수 있도록 제작하는 웹 기반 작업 도구입니다.

## 프로젝트 목적

React와 TypeScript를 활용해 제작 도구 형태의 사용자 인터페이스를 구현하고,
이미지 편집 상태와 비동기 생성 요청 상태를 관리하는 경험을 쌓기 위해 시작했습니다.

## 기술 스택

- React
- TypeScript
- Vite
- Zustand
- ESLint

## 주요 기능

- 이미지 드래그 앤 드롭 업로드
- 업로드 이미지 목록 관리
- 작업 영역 내 이미지 선택 및 편집
- 프롬프트와 이미지 생성 옵션 설정
- Mock 이미지 생성 요청
- 로딩, 성공, 실패 및 재시도 상태 처리
